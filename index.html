So funktioniert die App

Diese App dient zur einfachen Erfassung täglicher Produktionsdaten. Die Daten werden lokal im Browser gespeichert und nicht automatisch versendet.

Ablauf

  1. Produktionsdaten eingeben
  2. Eintrag speichern
  3. Einträge prüfen
  4. CSV exportieren
  5. CSV-Datei per E-Mail zurücksenden

Wichtige Hinweise

  • Die App ist eine offene Test-App.
  • Bitte nur freigegebene Produktionsdaten eingeben.
  • Keine personenbezogenen Daten eintragen.
  • Die Daten bleiben lokal im Browser dieses Geräts.
  • Wenn Sie die App an einem anderen Gerät öffnen, sind die lokal gespeicherten Daten dort nicht automatisch vorhanden.

Optionale OEE-Daten

OEE-Daten müssen nur ausgefüllt werden, wenn OEE berechnet werden soll. Wenn keine OEE-Daten eingetragen werden, zeigt die App bei OEE-Kennzahlen n/a an.

CSV

Über CSV exportieren können die erfassten Daten heruntergeladen und anschließend per E-Mail zurückgesendet werden. Über CSV importieren können vorhandene CSV-Daten wieder geladen werden.

App zurücksetzen

Über App zurücksetzen können alle lokal gespeicherten Daten dieses Browsers gelöscht werden. Dieser Schritt kann nicht rückgängig gemacht werden.
